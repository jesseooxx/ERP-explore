## 7. 完整 SQL 查詢範例

### 7.1 查詢 284102 完整資料的 SQL

```sql
-- 1. 查詢產品基本資料
SELECT * FROM tdm01 WHERE da01 = '284102';

-- 2. 查詢產品供應商關係
SELECT * FROM tcm05 WHERE ce02 = '284102';

-- 3. 查詢產品多行描述
SELECT * FROM tdm02 WHERE db01 = '284102' ORDER BY db02, db03;

-- 4. 查詢產品標準 BOM
SELECT
    de01 AS [產品編號],
    de02 AS [組件編號],
    CAST(de03 AS VARCHAR) + ' / ' + CAST(de04 AS VARCHAR) AS [比例],
    de05 AS [供應商],
    de09 AS [主要零件],
    de15 AS [顯示序號]
FROM tdm05
WHERE de01 = '284102' AND de18 = 'Y'
ORDER BY de15;

-- 5. 計算訂單需求 (假設訂購 1000 PC)
DECLARE @OrderQty FLOAT = 1000;

SELECT
    de02 AS [組件編號],
    CAST((@OrderQty * de03 / de04) AS DECIMAL(10,2)) AS [需求數量],
    de05 AS [供應商],
    CASE de09 WHEN 'Y' THEN '主要' ELSE '次要' END AS [類型]
FROM tdm05
WHERE de01 = '284102' AND de18 = 'Y'
ORDER BY de15;

-- 6. 查詢供應商基本資料
SELECT ca01, ca02, ca03 FROM tcm01 WHERE ca01 IN ('B02', '2279', '02291');

-- 7. 查詢所有相關資料的聯合查詢
SELECT 'tdm01產品主檔' AS [來源], COUNT(*) AS [筆數] FROM tdm01 WHERE da01 = '284102'
UNION ALL
SELECT 'tdm02多行描述', COUNT(*) FROM tdm02 WHERE db01 = '284102'
UNION ALL
SELECT 'tdm05產品BOM', COUNT(*) FROM tdm05 WHERE de01 = '284102'
UNION ALL
SELECT 'tcm05供應商產品', COUNT(*) FROM tcm05 WHERE ce02 = '284102';
```

## 12. 編號命名規則總結

### 12.1 產品編號規則

```
284102 = 6位數字產品編號
- 28: 可能代表產品系列或年份
- 4102: 流水號
```

### 12.2 組件編號規則

```
[主產品編號]-[組件類型]

類型後綴:
- MAIN      = 主要零件
- BAG LABEL = 袋標籤
- CTN LABEL = 箱標籤
- LABEL     = 一般標籤
- BOX       = 紙盒
- CARTON    = 外箱
```

### 12.3 供應商編號規則

```
- B02   = 可能是測試供應商 (TEST)
- 2279  = 正式供應商 (宏钰)
- 02291 = 標籤供應商 (可能是台寶數位)

命名規則:
- 4位數字: 一般供應商
- 5位數字: 可能是子供應商或特殊分類
- B##: 可能是測試/特殊供應商
```
