## 15. 常用查詢範例

### 15.1 查詢產品完整資料

```sql
-- 查詢產品基本資料
SELECT da01, da02, da03, da04, da05,
       da07, da08, da09, da10, da12,
       da40, da41, da42, da55, da56
FROM tdm01
WHERE da01 = '284006';

-- 查詢產品供應商關係
SELECT ce011, ce02, ce04, ce05, ce06, ce07, ce12
FROM tcm05
WHERE ce02 = '284006'
ORDER BY ce04;

-- 查詢產品BOM
SELECT de01, de02, de03, de04, de05, de06, de09, de15
FROM tdm05
WHERE de01 = '284006' AND de18 = 'Y'
ORDER BY de15;
```

### 15.2 計算訂單BOM需求

```sql
DECLARE @ProductID VARCHAR(20) = '284006';
DECLARE @OrderQty FLOAT = 1000;

SELECT
    de02 AS [組件編號],
    CAST((@OrderQty * de03 / de04) AS DECIMAL(10,2)) AS [需求數量],
    de05 AS [供應商],
    CASE de09 WHEN 'Y' THEN '主要' ELSE '次要' END AS [類型]
FROM tdm05
WHERE de01 = @ProductID AND de18 = 'Y'
ORDER BY de15;
```
